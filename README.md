# 实验楼A、B、C、D栋校园网网络建设设计方案

## 项目简介

本项目为湖南中医药大学（HNUCM）实验楼 A、B、C、D 栋的校园网网络建设设计方案。项目旨在构建高速、稳定、安全、智能的网络环境，满足中医药教学、科研及管理对网络基础设施的迫切需求。

## 主要特性

- **高性能**：无线单用户速率 ≥100Mbps，核心层带宽 ≥100Gbps
- **高可靠性**：核心设备双机热备，链路冗余，网络可用性 ≥99.99%
- **全面安全**：防火墙、IPS、802.1X认证、数据加密，符合等保2.0要求
- **智能运维**：统一管理平台，自动化配置与故障告警
- **无线全覆盖**：室内外 AP 高密度部署，支持快速漫游（≤50ms 切换）

## 技术架构

- **三层结构**：核心层 – 汇聚层 – 接入层
- **核心设备**：华为 CloudEngine S12700E-8（双机热备）
- **汇聚设备**：华为 S6730-H48X6C
- **接入设备**：华为 S5735S-L48P4X-A / S5735S-L24P4S-A1
- **无线 AP**：华为 AirEngine5761-12W（高密）、AirEngine5760-10（普通）、AirEngine8760R-X1（室外）
- **安全设备**：华为 USG6650 防火墙
- **服务器**：华为 FusionServer 2288H
- **路由器**：华为 NE20E-S8

## 主要成果

- 实地勘测平面图及 AP/交换机选址
- 信道分配方案与 IP 地址规划
- 综合布线设计（光纤、网线、电力）
- 完整设备清单及报价清单
